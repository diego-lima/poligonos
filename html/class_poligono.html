◆ area()

float Poligono::area ( void  )

Calcula a área do polígono usando Shoelace.

Funciona para polígonos côncavos, convexos e até mesmo polígonos que se interceptam em si mesmos. No caso desses últimos, a função recursivamente subdivide o polígono em polígonos menores, que não se interceptam em si mesmos (denominados polígonos simples), e aí retorna a soma das áreas desses subpolígonos.

Returns
a área total do polígono
