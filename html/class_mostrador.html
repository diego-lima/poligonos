Detailed Description

A classe Mostrador.

Serve para exibição de um gráfico de área na tela que mostra o contorno do polígono.

Recebe o poligono a ser mostrado através de um Area Chart do QT.

os parâmetros argc e argv devem ser os mesmos passados na função main. "int main(int argc, char *argv[])".

A exibição é feita com o area chart do QT. Como o area chart não é o mais adequado para exibição de polígonos quaisquer, essa exibição é defeituosa.

Parameters
argcda main
argvda main
poligonoo poligono a ser mostrado
enquadrarse deseja arrastar o polígono para a origem

Código adaptado do exemplo 'areachart' do próprio website do Qt. site: https://doc.qt.io/qt-5.10/qtcharts-areachart-example.html Caso necessario: $ sudo apt install libgl1-mesa-dev
